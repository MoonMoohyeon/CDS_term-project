# 협동분산시스템

## 🗓️기간
2024.04.25. ~ 2024.06.11

## ⚙️기술
- Java
- kotlin

## 📖소개
사용자 간 공유되는 화이트보드를 개발합니다.

프론트는 kotlin, 백은 Java를 사용합니다.

## ✅기능
1. 클라이언트 접속 관리
2. 도형 및 텍스트 그리기
3. 그림 객체 정보 공유
4. 저장 및 불러오기 기능

## 👥역할
- 개인 프로젝트
